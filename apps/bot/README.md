## How To Use

```bash
# Clone this repository
$ git clone https://github.com/serudda/reward-system.git

# Go into the repository
$ cd reward-system

# Copy .env.example to .env and update the values accordingly
$ cp .env.example .env

# Install dependencies
$ pnpm install

# Run the app
$ pnpm run dev
```
